Given the 'indices' of an load/store operation where the memref is a result of a subview op, returns the indices w.r.t to the source memref of the subview op.

For example

%0 = ... : memref<12x42xf32> %1 = subview %0[arg0, arg1][][stride1, stride2] : memref<12x42xf32> to memref<4x4xf32, offset=?, strides=[?, ?]> %2 = load %1[i1, i2] : memref<4x4xf32, offset=?, strides=[?, ?]>

could be folded into

%2 = load %0[arg0 + i1 * stride1][arg1 + i2 * stride2] : memref<12x42xf32>
