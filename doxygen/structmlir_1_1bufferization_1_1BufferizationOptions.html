◆ isOpAllowed()

bool mlir::bufferization::BufferizationOptions::isOpAllowed ( Operation *  op) const
inline

Return whether the op should be bufferized or not.

If no filter is specified (hasFilter = false), every op will be bufferized. Otherwise, an op is bufferized if:

  • At least one ALLOW filter says true.
  • And, no DENY filter says true.
